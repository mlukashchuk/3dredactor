◆ lightingControl()

void LightingAndRenderingModule::lightingControl ( )

Управляет освещением в сцене.

Метод предоставляет возможность настройки различных типов источников света, включая точечные, направленные и областные источники света, а также настройку их параметров.

◆ postProcessing()

void LightingAndRenderingModule::postProcessing ( )

Осуществляет постобработку изображений после рендеринга.

Метод содержит инструменты для коррекции цвета, добавления эффектов глубины резкости, свечения и других визуальных эффектов после получения изображения.

void LightingAndRenderingModule::renderingTools ( )

Предоставляет инструменты для выполнения рендеринга.

Метод содержит средства для создания фотореалистичных и нереалистичных изображений, включая поддержку ведущих рендер-движков.
